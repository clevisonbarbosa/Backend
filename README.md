# Instalação

```bash
$ npm ci
```

# Dotenv

Crie um arquivo `.env` baseado no [`.env.example`](./.env.example)

# Execução

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run build
$ npm run start:prod
```

# Api

Ir para a página http://localhost:8080/api
